# Currículum Vitae Académico — José Federico Castro Tramontina

Proyecto HTML/CSS preparado para:

- visualización en navegador;
- lectura por sistemas ATS;
- exportación a PDF en formato A4;
- publicación en español e inglés;
- presentación completa de antecedentes académicos, científicos, docentes y profesionales.

## Uso

Abrir `index.html` en Firefox, Chromium o Google Chrome y seleccionar el idioma correspondiente.

Para exportar:

1. Abrir la opción **Imprimir**.
2. Elegir **Guardar como PDF**.
3. Papel: **A4**.
4. Escala: **100 %**.
5. Márgenes: **ninguno**.
6. Activar **gráficos de fondo**.
7. Desactivar encabezados y pies de página del navegador.

## Observación

Este proyecto corresponde a la versión académica completa del currículum vitae. Su contenido se conserva de forma textual, sin resumir antecedentes ni establecer un límite de páginas.

Los datos de contacto, perfiles académicos, publicaciones, proyectos, instituciones y referencias externas aparecen como texto visible además de estar enlazados. Esto mejora la accesibilidad, la lectura por sistemas ATS y la utilidad del PDF impreso.
